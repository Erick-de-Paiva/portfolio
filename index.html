Loja Virtual de Informática (Front-End)

Projeto acadêmico desenvolvido em grupo, com foco na criação de um e-commerce front-end utilizando HTML, CSS e JavaScript. Fiquei responsável principalmente pela implementação do sistema de cadastro e login, incluindo validações de CPF, preenchimento automático de endereço via CEP, máscara para telefone e persistência de dados com LocalStorage.

O site conta com página principal com carrossel dinâmico, menu responsivo no formato hambúrguer com subdivisões e navegação entre páginas de produtos.

O projeto teve como objetivo aplicar conceitos de manipulação do DOM, validação de formulários e organização de uma aplicação web funcional.

Tecnologias: HTML, CSS e JavaScript.
